# About
Quiz for countries and capitas with quiz
# How to use
### Initilize
``` sh 
# Download the dependency
npm install
```
### Devlop
``` sh
# Live Build ReScript
npm run res:dev
```
``` sh
# Live Build (main)
npm run dev
```

### Build
``` sh
# Build ReScript
npm run res:build
# Build (main)
npm run build
# Deploy the "target" dir
```
